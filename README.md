# CS-IT-08-Project
This is our GitHub, used to upload the code from our 8th semester PBL(problem based learning) project about RL(reinforcement learning), more specificaly about investigating differentstrategies and methods that can be used to maximize the success rate of an agent whilst evaluating system reliability.
The Algorithms investigated are RainbowDQN, PPO, PLR, ASHA and PB2.
